# 🚀 AI Resume Analyzer SaaS

A smart AI-powered Resume Analyzer that evaluates resumes against job descriptions using NLP and Machine Learning. It provides ATS score, skill matching, job role prediction, and hire probability in a modern Streamlit web app.

## 📌 Project Overview

This project is an AI-based Resume Screening System that helps candidates understand how well their resume matches a job description. It simulates a real Applicant Tracking System (ATS) used in companies.

## 🎯 Features

- Upload Resume (PDF)
- Automatic Text Extraction
- Skill Detection Engine
- ATS Score Calculation
- Semantic Job Matching using AI
- Job Role Prediction
- Hire Probability Score
- Visual Dashboard with Charts

## 🧠 How It Works

Resume PDF → Text Extraction → Skill Extraction → AI Matching → ATS Score → Role Prediction → Hire Probability → Dashboard Output

## 🛠️ Tech Stack

Python, Streamlit, pdfplumber, scikit-learn, sentence-transformers, matplotlib, NLP

## 🚀 How to Run

cd AI-Resume-SaaS

python -m venv venv
venv\Scripts\activate

pip install -r requirements.txt

streamlit run app.py
