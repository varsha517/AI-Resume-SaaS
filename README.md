# 🚀 AI Resume Intelligence SaaS

> The next-generation AI-powered Resume Analyzer that evaluates resumes using ATS scoring, semantic AI matching, job role prediction, skill gap detection, and hire probability estimation — all inside a modern SaaS dashboard.

---

## 🌐 Product Vision

AI Resume Intelligence SaaS is a smart recruitment simulation system that replicates real-world Applicant Tracking Systems (ATS) used by top companies.

It helps candidates understand resume quality, improve skills, and increase selection chances using AI.

---

## ❌ Problem

Most candidates:
- Don’t know why they get rejected
- Fail ATS screening systems
- Lack feedback on resumes
- Apply without understanding job fit

---

## ✅ Solution

This AI-powered system solves it by:
- Giving instant ATS score
- Predicting job role fit
- Showing skill gaps
- Estimating hire probability

---

## ⚡ Features

- 📄 Upload Resume (PDF)
- 🧠 AI ATS Scoring Engine
- 🔍 Semantic Job Matching (AI embeddings)
- 🎯 Job Role Prediction Model
- 💼 Hire Probability Calculator
- 📊 Skill Gap Detection System
- 📑 Resume Comparison Mode
- 📈 Interactive Analytics Dashboard
- 📥 PDF Report Generation

---

## 🧠 How It Works

Resume Upload → PDF Text Extraction → Skill Detection → Semantic AI Matching → ATS Score Calculation → Role Prediction → Hire Probability → Dashboard Output

---

## 🛠️ Tech Stack

Python, Streamlit, Sentence Transformers, Scikit-learn, pdfplumber, Matplotlib, ReportLab, NLP

---

## 🧩 System Architecture

Frontend (Streamlit UI) → Backend (Python Engine) → AI Layer (Sentence Transformers Model) → Logic Layer (ATS Scoring System) → Export Layer (PDF Generator)

---

## 🚀 Setup & Run

git clone https://github.com/varsha517/AI-Resume-SaaS.git  
cd AI-Resume-SaaS  

python -m venv venv  
venv\Scripts\activate  

pip install -r requirements.txt  

streamlit run app.py
